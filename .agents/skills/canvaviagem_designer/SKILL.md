# Canvaviagem Designer — Skill Mestra

Use essa skill toda vez que o objetivo for criar uma postagem completa para o Instagram do @canvaviagem. Ela define a identidade visual e coordena as demais skills do ecossistema.

## Para quem é esse conteúdo

Você está criando para **agentes de viagem e donos de agência** — profissionais que trabalham muito, têm pouco tempo, não sabem o que postar e estão cansados de depender de indicação para conseguir cliente. O visual precisa parar o dedo deles no scroll e fazer eles sentirem que existe uma solução feita especificamente para eles.

## Identidade Visual

A estética do @canvaviagem é escura, premium e moderna — como se fosse uma plataforma tech de viagens de luxo. O fundo é sempre preto ou preto azulado. Os destaques são em azul vibrante e roxo suave. As fotos dos destinos são ultra-realistas e cinematográficas. A tipografia é bold e direta.

**Sobre a logo:** o ícone do @canvaviagem fica no canto superior esquerdo de cada post. É um ícone arredondado com gradiente azul e roxo. Ao lado dele vai o texto "@canvaviagem" em branco.

**Paleta de cores:**
- Fundo principal: preto profundo
- Fundo alternativo: preto com tom azulado
- Azul vibrante para destaques e palavras-chave
- Roxo suave para glow, sombras e efeitos
- Branco para o texto principal
- Cinza claro para o subtexto
- Azul para os botões de CTA

## Fluxo para Criar uma Postagem Completa

Siga essa ordem:
1. Use a skill **Pesquisa de Mercado** para entender o contexto antes de criar qualquer coisa
2. Use a skill **Copywriter** para definir o tema e escrever o texto
3. Use a skill **Stories** ou **Feed** para gerar a imagem
4. Use a skill **Revisor** para validar antes de publicar

## Skills do Ecossistema

- **Pesquisa de Mercado** — entende concorrentes, gargalos e diferenciais antes de criar
- **Stories Creator** — imagens no formato 9 por 16 (Stories e Reels)
- **Feed Creator** — imagens no formato 4 por 5 (Feed do Instagram)
- **Copywriter** — textos e legendas de alta conversão
- **Revisor** — aprovação final da postagem completa

## O Produto

O @canvaviagem vende um pack com 150 vídeos prontos para agências de viagem postarem no Instagram — por R$47. O agente compra, pega os vídeos, publica e aparece. Sem precisar gravar nada, editar nada ou contratar ninguém. Esse é o produto principal e ele precisa aparecer em toda postagem de venda.
