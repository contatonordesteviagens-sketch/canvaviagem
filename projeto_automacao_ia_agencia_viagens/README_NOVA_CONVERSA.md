# Automacao IA para Agencia de Viagens

Este e um projeto separado do Canva Viagem.

Objetivo: vender e operar uma automacao com agentes de inteligencia artificial para uma agencia de viagens, cobrindo marketing organico, remarketing, WhatsApp, email, SEO, artes, videos, landing pages, planilhas, gestao comercial e relatorios.

## Como iniciar uma nova conversa

Cole este prompt no inicio da nova conversa:

```text
Voce esta dentro do projeto AUTOMACAO_IA_AGENCIA_DE_VIAGENS.

Este projeto nao e o Canva Viagem. Ele usa agentes e skills ja existentes na maquina para montar uma automacao vendavel para uma agencia de viagens.

Objetivo do projeto:
- Criar postagens automaticas para Instagram, TikTok, Reels e Stories.
- Criar artes, briefings visuais, prompts de imagem e criativos.
- Criar roteiros de videos e Reels.
- Criar sequencias de WhatsApp, email e remarketing.
- Criar estrategia de SEO, blog e paginas.
- Criar planilhas de vendas, funil, entradas, saidas e follow-up.
- Criar relatorios e rotina semanal de gestao.

Sempre que eu pedir uma entrega, escolha o departamento certo, cite os agentes usados e entregue algo pronto para usar.

Departamentos principais:
1. Diretoria de IA
2. Marketing Organico
3. Artes e Criativos
4. Videos e Reels
5. Remarketing WhatsApp/Email
6. SEO e Autoridade
7. Planilhas e Gestao Comercial
8. Midia Paga e Remarketing Pago
9. Automacao e Integracoes

Use os arquivos deste projeto como base:
- AGENTES_INVENTARIO_COMPLETO.csv
- RESUMO_INVENTARIO_AGENTES.md
- docs/PAINEL_REAL_AGENTES_IA_AGENCIA_VIAGENS.md
- MANUAL_ATIVACAO_AGENTES.md
- PLAYBOOK_ENTREGAVEIS_TANGIVEIS.md
- PROMPTS_DE_PRODUCAO.md
```

## Arquivos principais

- `AGENTES_INVENTARIO_COMPLETO.csv`: inventario gerado dos agentes/skills encontrados.
- `RESUMO_INVENTARIO_AGENTES.md`: resumo quantitativo do inventario.
- `MANUAL_ATIVACAO_AGENTES.md`: como ativar e usar cada grupo de agentes.
- `PLAYBOOK_ENTREGAVEIS_TANGIVEIS.md`: entregaveis reais que podem ser criados para o cliente.
- `PROMPTS_DE_PRODUCAO.md`: comandos prontos para gerar materiais.
- `docs/`: materiais anteriores, painel HTML e roteiro de reuniao.
