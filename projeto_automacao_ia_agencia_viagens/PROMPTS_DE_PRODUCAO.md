# Prompts de producao

## Diagnostico

```text
Use Canvaviagem Orquestrador, Canvaviagem CEO e Canvaviagem Diretor de Marketing. Diagnostique uma agencia de viagens com os seguintes dados: [cole dados]. Entregue gargalos, oportunidades, prioridade da semana, departamentos acionados e plano de 7 dias.
```

## Post do dia

```text
Use Departamento de Marketing Social e Canvaviagem Copywriter. Crie o post do dia para uma agencia de viagens vender [destino]. Entregue formato, hook, legenda, CTA, imagem sugerida, stories de apoio e objetivo comercial.
```

## Carrossel

```text
Use marketing-carousel-growth-engine e Canvaviagem Copywriter. Crie um carrossel de 8 slides sobre [tema] para atrair leads de viagem. Entregue titulo de cada slide, texto curto, direcao visual e CTA final para WhatsApp.
```

## Reel

```text
Use Departamento de Videos & Reels e marketing-short-video-editing-coach. Crie um Reel de 30 segundos sobre [tema/destino]. Entregue hook, cenas, texto na tela, narracao, cortes, B-roll, legenda e CTA.
```

## Story

```text
Use Canvaviagem Stories Creator. Crie uma sequencia de 7 stories para vender [pacote]. Inclua enquete, caixa de pergunta, prova social, oferta, escassez honesta e CTA para WhatsApp.
```

## Arte

```text
Use Canvaviagem Feed Creator, Canvaviagem Designer e design-image-prompt-engineer. Crie uma arte de feed 4:5 para vender [destino]. Entregue texto principal, subtitulo, CTA, direcao visual, prompt de imagem e variacao A/B.
```

## Imagem realista

```text
Crie um prompt de imagem ultra-realista para uma campanha de agencia de viagens vendendo [destino]. A imagem deve ter [publico], [cenario], [emocao], espaco para texto, formato [4:5 ou 9:16] e aparencia premium.
```

## WhatsApp follow-up

```text
Use Canvaviagem Atendimento e sales-discovery-coach. Crie um follow-up de WhatsApp para um lead que pediu orcamento de [destino] ha [tempo] e nao respondeu. Entregue 5 mensagens progressivas, tom humano e CTA.
```

## Email remarketing

```text
Use Canvaviagem Diretor de Email Marketing e Canvaviagem Aquisicao Email. Crie 4 emails para leads interessados em [destino] que nao fecharam. Entregue assunto, preheader, corpo, CTA e objetivo de cada email.
```

## SEO

```text
Use Canvaviagem SEO Manager e marketing-seo-specialist. Crie uma pauta SEO para [destino/cidade]. Entregue palavra-chave principal, secundarias, titulo, meta description, estrutura H2/H3 e CTA para WhatsApp.
```

## Landing page

```text
Use landing-page-dr, copy-direct-response e design-ui-designer. Crie uma landing page de alta conversao para vender [pacote]. Entregue secao por secao, copy, CTA, provas, FAQ e direcao visual.
```

## Planilha

```text
Use Spreadsheets, sales-pipeline-analyst e support-finance-tracker. Crie uma planilha de gestao comercial para agencia de viagens. Entregue abas, colunas, formulas, status do funil e rotina semanal.
```

## Relatorio

```text
Use Canvaviagem Relatorio Visual e support-analytics-reporter. Crie um modelo de relatorio semanal para agencia de viagens acompanhar conteudo, leads, vendas, follow-ups, campanhas e proximas acoes.
```
